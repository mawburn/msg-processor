# Message Processor

To run the project for the first time:

```
yarn install

cd client
yarn install

cd ..
yarn start
```

Production Build: 

```
cd client
yarn build
```

Dev Server: http://localhost:3000

Live Demo: https://msg-processor.herokuapp.com/
